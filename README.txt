Super Web Share helps to easily add native share action to your website.

== Description ==

Super Web Share plugin for WordPress helps to increase your traffic and engagement by a single quick click to share the website with your friends and social medias. Super Web Share works on all supported browsers like Chrome for Android, Edge for Android, Brave for Android and also on Opera for Android. It will show the native apps which are installed upon the device to share the article with others, and due to this users can easily share the page/post with other's easily in just a single click. Upon the native share which prompts up, the users can also copy the link of the website easily.

It takes less than a minute to setup Super Web Share, and we provide a clean uninstall by removing the entire data entry it creates. We also won't save any settings to your database until you manually save the settings.

Currently Super Web Share plugin provides a floating button and can set the color, Position of the floating button and can set the pages which you would like to show via our Settings page to easily add the share feature to your websites.

#### Want to know how SuperWebShare works? 

* Open [JoseVarghese.com](https://www.josevarghese.com) in a supported device. 
* You can see a floating button at the right bottom corner
* Tap on it, instantly you can see the Web Share prompt with the applications which you can share the page
* Tap on the icon of application from the prompt, and send it. 

#### Requirments for working SuperWebShare?
* Website should be served fully via a secured origin, ie; green padlock should be there on the address for working Web Share

== Installation ==

* Visit WordPress Admin Dashboard > Plugins > Add New
* Search for 'Super Web Share'
* Click "Install now" and then "Activate" Super Web Share

Install manually:
1. Upload `super-web-share.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Customize Super Web Share ==
You can easily customize the color, position and pages which you would like to show the floating share button. To customize goto Settings >> Super Web Share, from the page you can easily customise the floating share button.

== Frequently Asked Questions ==

= Upon which all browsers does the Super Web Share plugin works =

The plugin works on browsers like Chrome for Android, Edge for Android, Brave for Android and also on Opera for Android

= Will the floating share button shows upon non-supported browsers=

No, the floating share button will be hided automatically, if the browser won't support Web Share API
